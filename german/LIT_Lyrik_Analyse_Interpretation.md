# 6. Satzanfänge für Gedichtanalysen

## Einleitung
- Das Gedicht „…“ von … thematisiert …
- Im Mittelpunkt des Gedichts steht …
- Das lyrische Ich beschreibt …

## Inhalt
- Zu Beginn des Gedichts …
- Im weiteren Verlauf …
- Anschließend …
- Gegen Ende des Gedichts …
- Abschließend wird deutlich, dass …

## Analyse
- Besonders auffällig ist …
- Dies wird deutlich in …
- Ein Beispiel dafür findet sich in Vers …
- Dadurch wird verdeutlicht, dass …
- Dies unterstreicht …

## Interpretation
- Daraus lässt sich schließen, dass …
- Dies könnte darauf hindeuten, dass …
- Insgesamt wird deutlich, dass …

## Schluss
- Zusammenfassend lässt sich sagen, dass …
- Abschließend lässt sich festhalten, dass …
- Die Analyse zeigt, dass …

---

# 7. Beispiel-Sätze (B2–C1 Niveau)

Die Metapher „…“ verdeutlicht die Überforderung des lyrischen Ichs.

Durch die bildhafte Sprache entsteht eine düstere Atmosphäre.

Die Wiederholung des Wortes „…“ unterstreicht die zentrale Bedeutung dieses Motivs.

Im weiteren Verlauf des Gedichts verändert sich die Stimmung deutlich.

Zusammenfassend lässt sich feststellen, dass das Gedicht eine kritische Haltung gegenüber dem Stadtleben zum Ausdruck bringt.

---

# 8. Struktur einer Gedichtanalyse (Kurzüberblick)

1. Einleitung  
Autor, Titel, Thema, Deutungshypothese  

2. Inhalt  
Entwicklung im Gedicht, Gefühle des lyrischen Ichs  

3. Analyse  
sprachliche Mittel, Form (Reim, Metrum, Kadenz)  

4. Schluss  
Interpretation und Aussage des Gedichts
