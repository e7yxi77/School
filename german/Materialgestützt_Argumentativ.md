# Materialgestütztes argumentatives Schreiben – Kurz-Zusammenfassung (ZKA Deutsch)

## Ziel
Ein **materialgestützter argumentativer Text** nutzt Materialien (Texte, Statistiken, Grafiken),  
um **eine begründete eigene Position zu einem Thema darzustellen und zu argumentieren**.

Du musst:
- Materialien **auswerten**
- **Argumente entwickeln**
- **eine eigene Position vertreten**
- **logisch argumentieren**

---

# 1. Struktur

## Einleitung
- Thema vorstellen
- Problem erklären
- evtl. Bedeutung des Themas zeigen

**Satzanfänge**

- In den letzten Jahren wird häufig über … diskutiert.
- Das Thema „…“ spielt in der heutigen Gesellschaft eine wichtige Rolle.
- Immer mehr Menschen beschäftigen sich mit der Frage, ob …
- Der folgende Text setzt sich mit der Frage auseinander, ob …

---

## Hauptteil

Im Hauptteil stellst du **Argumente dar und nutzt die Materialien**.

Typische Struktur:

### Argument 1
These → Erklärung → Beispiel / Material

### Argument 2
These → Erklärung → Beispiel / Material

### Argument 3
These → Erklärung → Beispiel / Material

Du kannst auch **Gegenargumente** erwähnen und danach entkräften.

---

# Materialien einbauen

- Laut Material M1 …
- In Material M2 wird deutlich, dass …
- Die Grafik in M3 zeigt, dass …
- Aus Material M4 geht hervor, dass …
- Die Statistik verdeutlicht …

---

# Argumente formulieren

## Ein Argument einführen

- Ein wichtiger Grund dafür ist …
- Ein weiteres Argument ist …
- Außerdem spricht dafür, dass …
- Darüber hinaus lässt sich feststellen, dass …

## Beispiele geben

- Zum Beispiel …
- Beispielsweise zeigt Material M2, dass …
- Ein Beispiel dafür ist …

---

# Gegenargumente

- Allerdings wird häufig argumentiert, dass …
- Kritiker behaupten, dass …
- Ein möglicher Einwand ist …

**Entkräften**

- Dennoch zeigt sich, dass …
- Allerdings überzeugt dieses Argument nicht vollständig, da …
- Dennoch überwiegen die Vorteile, weil …

---

# Schluss

- eigene Position zusammenfassen
- Ergebnis formulieren

**Satzanfänge**

- Zusammenfassend lässt sich sagen, dass …
- Insgesamt zeigt sich, dass …
- Abschließend lässt sich festhalten, dass …
- Daher ist es sinnvoll, …

# 4. Beispiel-Sätze

Ein wichtiger Grund für diese Entwicklung ist …

Darüber hinaus zeigt Material M2, dass …

Ein mögliches Gegenargument ist, dass …

Dennoch überwiegen die Vorteile, da …

Zusammenfassend lässt sich feststellen, dass …

---

# 5. Wichtige Tipps für ZKA

- Materialien **immer nennen (M1, M2, …)**  
- **eigene Argumente entwickeln**  
- **klare Struktur (Einleitung – Argumente – Schluss)**  
- **Gegenargumente einbauen**  
- **logisch und sachlich argumentieren**
