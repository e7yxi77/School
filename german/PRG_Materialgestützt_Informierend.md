# Informierender materialgestützter Text – Kurz-Zusammenfassung (ZKA Deutsch)

## Ziel
Ein **informierender materialgestützter Text** erklärt ein Thema **mithilfe von Materialien (Texte, Grafiken, Statistiken)**.  
Du **fasst Informationen zusammen, erklärst Zusammenhänge und verweist auf die Materialien**.  
Wichtig: **sachlich schreiben, keine eigene Meinung**.

---

# 1. Struktur

## Einleitung
- Thema vorstellen
- kurz erklären, worum es geht

**Satzanfänge**

- Das Thema „…“ gewinnt zunehmend an Bedeutung.
- In den letzten Jahren wird häufig über … diskutiert.
- Der folgende Text informiert über …
- Ziel dieses Textes ist es, … zu erläutern.

---

## Hauptteil
Hier erklärst du die Informationen aus den Materialien.

Typische Reihenfolge:

1. Definition / Hintergrund  
2. wichtige Fakten oder Ursachen  
3. Beispiele / Folgen / Entwicklungen  

**Materialien einbauen**

- Laut Material M1 …
- In Material M2 wird dargestellt, dass …
- Die Grafik in M3 zeigt, dass …
- Aus Material M4 geht hervor, dass …
- Die Statistik verdeutlicht …

---

## Schluss
Die wichtigsten Punkte kurz zusammenfassen.

**Satzanfänge**

- Zusammenfassend lässt sich sagen, dass …
- Insgesamt wird deutlich, dass …
- Abschließend zeigt sich, dass …
- Daraus lässt sich schließen, dass …

---

# 3. Gute Analyse-Verben

- erläutern  
- darstellen  
- erklären  
- verdeutlichen  
- veranschaulichen  
- aufzeigen  
- hervorheben  

---

# 4. Beispiel-Sätze

Die Grafik verdeutlicht, dass …

Aus Material M2 geht hervor, dass …

Ein weiterer wichtiger Aspekt ist …

Darüber hinaus zeigt sich, dass …

Zusammenfassend lässt sich feststellen, dass …

---

# 5. Wichtige Tipps für ZKA

- Materialien **immer nennen (M1, M2, …)**  
- **sachlich und neutral schreiben**  
- Informationen **nicht nur abschreiben, sondern erklären**  
- **klare Struktur (Einleitung – Hauptteil – Schluss)**
