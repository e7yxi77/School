# Mathe – Komplette Lösungen mit Rechenweg

## 1. Quadratische Funktionen

### 1. Scheitelpunkt von f(x) = x² - 6x + 5

f(x) = x² - 6x + 5  
= (x² - 6x + 9) - 9 + 5  
= (x - 3)² - 4  

Scheitelpunkt: S(3 | -4)

---

### 2. Nullstellen von f(x) = x² - 9

x² - 9 = 0  
(x - 3)(x + 3) = 0  

x₁ = 3  
x₂ = -3  

---

### 3. Scheitelpunktform von f(x) = 2x² - 8x + 1

2x² - 8x + 1  
= 2(x² - 4x) + 1  
= 2(x² - 4x + 4 - 4) + 1  
= 2((x - 2)² - 4) + 1  
= 2(x - 2)² - 8 + 1  
= 2(x - 2)² - 7  

---

### 4. Öffnung des Graphen

f(x) = -x² + 4x - 3  

Da a = -1 < 0, öffnet sich der Graph nach unten.

---

### 5. Scheitelpunkt S(2 | -3), Punkt (4 | 5)

Ansatz:  
f(x) = a(x - 2)² - 3  

5 = a(4 - 2)² - 3  
5 = 4a - 3  
8 = 4a  
a = 2  

f(x) = 2(x - 2)² - 3  

---

### 6. Scheitelpunkt S(-1 | 4), Punkt (1 | 0)

f(x) = a(x + 1)² + 4  

0 = a(2)² + 4  
0 = 4a + 4  
a = -1  

f(x) = -(x + 1)² + 4  

---

### 7. Drei Punkte (0 | 1), (1 | 4), (2 | 9)

Allgemeine Form: f(x) = ax² + bx + c  

(0 | 1): c = 1  

(1 | 4): a + b + 1 = 4 → a + b = 3  
(2 | 9): 4a + 2b + 1 = 9 → 2a + b = 4  

Subtraktion:  
(2a + b) - (a + b) = 4 - 3  
a = 1  
b = 2  

f(x) = x² + 2x + 1  

---

## Diskriminante

### 8. x² - 5x + 6 = 0

D = (-5)² - 4·1·6  
D = 25 - 24  
D = 1  

x = (5 ± 1)/2  

x₁ = 3  
x₂ = 2  

---

### 9. 2x² + 4x + 5 = 0

D = 4² - 4·2·5  
D = 16 - 40  
D = -24  

Keine reellen Lösungen.

---

### 10. x² - 4x + 4 = 0

D = 16 - 16 = 0  

Eine doppelte Lösung:  
x = 2  

---

### 11. Bedingung für zwei reelle Lösungen

ax² + 2x + 1 = 0  

D = 4 - 4a  

Für zwei verschiedene Lösungen:  
4 - 4a > 0  
a < 1
